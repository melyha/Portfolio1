# Summary
I am creating a professional, visually appealing portfolio website to showcase my UX/UI design work and capabilities through both content and design. I'm building it using HTML5, CSS3, JavaScript, and the Bootstrap framework, with 4-5 responsive pages including Home, Projects, Resume, Creative Corner, and Contact sections. My website aims to serve potential employers, clients, and collaborators by presenting my experience, process, and projects in a clean, navigable format. I'm implementing mobile-first responsive design principles, interactive elements via jQuery, custom styling beyond Bootstrap defaults, and professional aesthetics that reflect my design skills, with the entire project being version-controlled through Git.

# Summary
I followed a structured and iterative development process, beginning with establishing the fundamental structure before progressively adding features and refinements. I started with reviewing class notes and creating a basic HTML skeleton, emphasizing building a solid foundation first. I implemented content next, with styling and interactive elements added in subsequent phases. Bootstrap provided me significant advantages for responsive design implementation, though customizing its default styling required additional effort from me. Over-riding Bootstrap styling was really hard. Trying to achieve the results that I wanted, I had to work a lot on CSS to make it look how I wanted it to be. Due to time restrictions, I didn't take the Sass way because I also didn't have such good command on it, and I thought learning that at this crucial time wasn't the best approach, so I went with customized CSS and Bootstrap. Bootstrap also impacted the jQuery features that I was applying, especially on navigation, so yeah, that's the hard part. Challenges I encountered included responsive layout adjustments, time constraints limiting my feature implementation, and difficulties I faced integrating jQuery features with Bootstrap components. My iterative process allowed for continuous learning and adaptation, including insights I gained about resource integrity verification and effective content organization. This step-by-step approach facilitated my organized development while allowing me to incorporate feedback throughout my project lifecycle.
